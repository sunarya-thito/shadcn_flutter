## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `defaultStateOrder` | `List<WidgetState>` | Default state priority order for resolving multiple active states.  Defines the precedence when multiple widget states are active simultaneously. States earlier in the list take priority over later ones. The default order prioritizes accessibility and interaction feedback appropriately. |
