# MenuTheme

{@template menu_theme}
Styling options for menu widgets such as [MenuGroup] and [MenuButton].
{@endtemplate}

## Usage

### Basic Example
```dart
MenuTheme(
)
```



## Features
- Responsive design
- Customizable styling
- Accessibility support

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `itemPadding` | `EdgeInsets?` | Default padding applied to each menu item. |
| `subMenuOffset` | `Offset?` | Offset applied when showing a submenu. |
