# GroupWidget

A multi-child layout widget that positions children using absolute coordinates.

## Usage

### Basic Example
```dart
GroupWidget(
  children: [
    GroupPositioned(
      top: 10,
      left: 10,
      child: Text('Positioned text'),
    ),
  ],
)
```



## Features
- Responsive design
- Customizable styling
- Accessibility support
