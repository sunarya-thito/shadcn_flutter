# EyeDropperLayer

A layer widget that provides eye dropper (color picking) functionality.

## Usage

No examples found for eye_dropper.

## Features
- Responsive design
- Customizable styling
- Accessibility support

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `child` | `Widget` | The child widget to wrap. |
| `previewAlignment` | `AlignmentGeometry?` | Alignment of the preview overlay. |
| `showPreview` | `bool` | Whether to show the magnified preview. |
| `previewSize` | `Size?` | Size of the preview overlay. |
| `previewScale` | `double` | Magnification scale of the preview. |
| `previewLabelBuilder` | `PreviewLabelBuilder?` | Builder for custom preview label widgets. |
