# MenuPopupTheme

A theme for [MenuPopup].

## Usage

### Basic Example
```dart
MenuPopupTheme(
)
```



## Features
- Responsive design
- Customizable styling
- Accessibility support

## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `surfaceOpacity` | `double?` | The opacity of the surface. |
| `surfaceBlur` | `double?` | The blur applied to the surface. |
| `padding` | `EdgeInsetsGeometry?` | The padding inside the popup. |
| `fillColor` | `Color?` | The background color of the popup. |
| `borderColor` | `Color?` | The border color of the popup. |
| `borderRadius` | `BorderRadiusGeometry?` | The border radius of the popup. |
