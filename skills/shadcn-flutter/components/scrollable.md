## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `diagonalDragBehavior` | `DiagonalDragBehavior?` | Behavior for diagonal drag gestures. |
| `dragStartBehavior` | `DragStartBehavior?` | When drag gestures should start. |
| `keyboardDismissBehavior` | `ScrollViewKeyboardDismissBehavior?` | How the keyboard dismissal should behave. |
| `clipBehavior` | `Clip?` | How to clip the scrollable content. |
| `hitTestBehavior` | `HitTestBehavior?` | How hit testing should behave for the scrollable. |
| `overscroll` | `bool?` | Whether overscroll effects are enabled. |
