## Properties

| Property | Type | Description |
| :--- | :--- | :--- |
| `checkboardPrimary` | `Color` | Primary color for the checkerboard pattern. |
| `checkboardSecondary` | `Color` | Secondary color for the checkerboard pattern. |
| `checkboardSize` | `double` | Size of each checkerboard square. |
