# shadcn_flutter monorepo

## Packages

- [`packages/shadcn_flutter/`](packages/shadcn_flutter/) — the published
  library ([README](packages/shadcn_flutter/README.md), [pub.dev](https://pub.dev/packages/shadcn_flutter))
- [`packages/shadcn_flutter_genui/`](packages/shadcn_flutter_genui/) — a
  [GenUI](https://docs.flutter.dev/ai/genui) catalog that renders AI-generated
  interfaces using shadcn_flutter widgets
- [`packages/docs/`](packages/docs/) — the Flutter Web docs/component gallery
  ([sunarya-thito.github.io/shadcn_flutter](https://sunarya-thito.github.io/shadcn_flutter/))
- [`packages/shadcn_flutter/example/`](packages/shadcn_flutter/example/) — a minimal consumer app
- [`packages/gen/`](packages/gen/) — developer tooling (icon/style/color
  generators, docs and LLM reference generators)

## Getting started

```shell
flutter pub get   # resolves every package in the workspace at once
flutter analyze
```
